# Security Policy

## Supported Versions

We actively support the latest version of Task Anchor. Please ensure you are running the most recent release before reporting a vulnerability.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Prev  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within Task Anchor, please do **not** report it via the public issue tracker. Instead, follow these steps:

1.  **Email Us:** Send a detailed report to [security@kodaxadev.com](mailto:security@kodaxadev.com).
2.  **Include Details:** Provide as much information as possible, including steps to reproduce, the impact of the vulnerability, and any potential fixes.
3.  **Wait for Response:** We will acknowledge your report within 48 hours and provide an estimated timeline for a fix.
4.  **Disclosure:** We ask that you give us a reasonable amount of time to resolve the issue before making any info public.

Thank you for helping keep Task Anchor secure for everyone.
