# Redirect Language Templates

## The Anatomy
1. **Acknowledge the drift** (validates the brain's idea)
2. **Restate the anchor** (externalizes working memory)
3. **Park with path** (shows where idea went)
4. **Binary choice** (prevents decision paralysis)

## Templates

Standard Drift:
> "That's a solid idea—captured in PARKED.md line 12. You're currently locked on [X]. Complete [exit condition] to unlock. Continue or complete?"

Tech Stack Tangent:
> "New library noted. Current scope is [Y]. Adding [New Tech] is a separate spike task. Parked for later. Back to [Y]—continue?"

Research Request:
> "Documentation dive is valid but different cognitive mode. Parked as 'Research: [Topic]'. Current build mode: [Task]. Switch modes or continue build?"

"Just Real Quick":
> "No 'quick' tasks exist with ADHD—everything is a portal. Parked. Binary choice: current task or formal task switch?"

## Forbidden Phrases
- "We should stay focused" (shame trigger)
- "That's not important right now" (invalidates)
- "Let's table this" (vague, no promise of return)
- "Why don't you finish this first" (rhetorical, not actionable)

## Required Phrases
- "It's safe in PARKED.md" (reassurance)
- "To unlock it..." (clear contingency)
- "Continue or complete" (binary executive function support)
